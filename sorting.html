This is a program I made to implement bubble sort, insertion sort, selection sort, merge sort and quick sort (pivot = first index) algorithms. The purpose is this program is to compute the CPU processing time for all the algorithms for varying input sizes as follows: N = 102, 103, 104, 105, and 106, using a random number generator to generate the inputs. Obtain the inputs from the following input ranges: 1- 103, 1 - 106, 1 – 109, 1 – 1012. The result graph can be found on my github page.
